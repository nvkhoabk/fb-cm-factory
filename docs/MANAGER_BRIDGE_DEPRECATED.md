# Manager Bridge Deprecated

Manager V1 Bridge is deprecated.

Host Agent V2 direct execution is the preferred path for real worker testing and production automation. New execution paths should use Host Agent V2, Script Runtime, Runtime Sessions, and dynamic instance allocation.

The `manager-bridge` backend module remains available only for backward compatibility with existing Manager V1 integrations. Do not delete existing APIs, but do not build new UI flows or job executor paths that encourage Manager V1 Bridge usage.
